# 小鹏的日语笔记 - AI日语口语训练平台

## 项目简介

这是一个基于AI技术的现代化日语学习平台，专注于提升用户的日语听说能力。网站采用清新简约的日式设计风格，结合游戏化学习元素，为用户提供沉浸式的日语学习体验。

## 🎯 核心功能

### 1. 用户认证系统
- 用户注册/登录
- 个人学习档案管理
- 学习进度追踪

### 2. 首页功能系统
- 智能学习提醒
- 今日学习任务展示
- 快速功能入口

### 3. 单词库系统
- 主题分类词汇库
- JLPT分级词汇
- 智能复习算法

### 4. 听力训练系统
- 隔空投送式听力练习
- JLPT真题听力
- 分级听力材料

### 5. AI助教系统
- 智能对话练习
- 语法纠错指导
- 发音评估反馈

### 6. AI口语外教（核心功能）
- 24/7实时语音对话
- 情境对话练习
- 智能发音评估

### 7. JLPT备考系统
- 真题下载资源
- 模拟考试系统
- 成绩分析报告

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome 80+, Firefox 75+, Safari 13+）
- 本地服务器环境（推荐使用Live Server）

### 安装步骤
1. 克隆项目到本地
```bash
git clone [项目地址]
cd ai日语口语训练
```

2. 使用Live Server启动项目
```bash
# 如果使用VS Code，安装Live Server扩展
# 右键index.html选择"Open with Live Server"
```

3. 在浏览器中打开 `http://localhost:5500`

## 🔧 自定义配置

### 颜色主题
修改 `styles.css` 中的CSS变量：
```css
:root {
    --primary-color: #4a90e2;      /* 主色调 */
    --secondary-color: #f39c12;    /* 辅助色 */
    --success-color: #27ae60;      /* 成功色 */
    --danger-color: #e74c3c;       /* 危险色 */
    --warning-color: #f1c40f;      /* 警告色 */
}
```

### 字体设置
```css
body {
    font-family: 'Noto Sans SC', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}
```

### 动画时长
```css
:root {
    --transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```
